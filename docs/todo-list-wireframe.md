# Todo List Wireframe

```
-----------------------------
|        Todo List         |
-----------------------------
| [新增待辦輸入框] [新增]   |
-----------------------------
| [全部] [已完成] [未完成]  |
-----------------------------
| [ ] 事項1        [編輯] [刪除] |
| [x] 事項2        [編輯] [刪除] |
| ...                        |
-----------------------------
```

- [ ] 代表未完成， [x] 代表已完成
- [編輯]、[刪除] 為操作按鈕
- 上方有篩選按鈕
